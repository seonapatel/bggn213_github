GGPLOT

The main function in the ggplot2 package is ggplot() first I need to install the ggplot2 package. I can install any package with the functioninstall.packages() > N.B. Never want to run install.packages() in my quarto document because it will install it every time you render the page. Instead, install it in the console so you only have to do it once.

library(ggplot2)
ggplot(cars) +
  aes(x=speed, y=dist) +
  geom_point()
ggplot(cars) +
  aes(speed) +
  geom_histogram()
`stat_bin()` using `bins = 30`. Pick better value `binwidth`.

Every ggplot needs at least 3 things:

  • The data (given with ggplot(cars))
  • The aesthetic mapping (given with aes())
  • The geom (given by geom_point())

For simple canned graphs “base” R is nearly always faster (in writing code and running)
